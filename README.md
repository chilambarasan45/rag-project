# RAG Document QA System

This project implements a Retrieval-Augmented Generation (RAG) system to answer questions from PDF documents.

## Features
- PDF loading and text extraction
- Text chunking
- FAISS vector database
- Semantic retrieval
- Transformer-based answer generation

## Tech Stack
- Python
- LangChain
- FAISS
- HuggingFace Transformers

## How to Run
1. Install dependencies
2. Run main.py
3. Ask questions from the document

## Example
Q: What is the report about?
A: It is a comprehensive report including text, images, and tables.
